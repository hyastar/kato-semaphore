对，用 conda 创建独立环境是最好的做法。直接执行：

```bash
conda create -n kato-semaphore python=3.12 -y
conda activate kato-semaphore
```

然后安装依赖，后端只需装一次就够（两个后端用的包完全一样）：

```bash
cd G:\Cursor\kato-semaphore
pip install -r requirements.txt
```

额外装训练需要的包（requirements.txt 里没有）：

```bash
pip install matplotlib seaborn
```

装完之后验证一下：

```bash
python -c "import fastapi, numpy, scipy, sklearn, joblib; print('全部OK')"
```

---

**Cursor 里配置解释器**

在 Cursor 按 `Ctrl+Shift+P` → 输入 `Python: Select Interpreter` → 选择 `kato-semaphore` 那个环境，路径类似：

```
C:\Users\你的用户名\anaconda3\envs\kato-semaphore\python.exe
```

这样 Cursor 的 AI 补全和终端都会用这个环境，不会串到别的项目。
